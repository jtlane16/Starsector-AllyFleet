# Starsector Ally Fleet Mod

Create permanent AI ally fleets that behave like independent captains with their own ships, resources, money, and AI logic.

## Features

- **Create Ally Fleets** at any station — choose the fleet name and initial composition
- **Priority System** — configure what your ally does: defend colonies, follow you, trade, or patrol
- **Persistent Economy** — allies have their own credits, supplies, fuel, and crew
- **Custom AI** — allies make their own decisions based on priorities and circumstances
- **Respawn** — allies rebuild their fleet at their home station if defeated
- **Intel UI** — monitor and command your ally fleets from the intel tab

## Installation

1. Copy `AllyFleet` folder to `Starsector/mods/`
2. Enable in the launcher

## Building

Requires the Starsector API (`starfarer.api.jar`) from the game installation.

```
ant build
```
